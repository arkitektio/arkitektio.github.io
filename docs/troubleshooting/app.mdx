First, we are sorry to hear that you are having issue. As an open source project in an early developmental stage, we are aware that there are still many bugs and issues to be fixed.

# RESTART EVERYTHING

It sounds stupid, but this is easily the easiest fix for most issues. If you encounter any issues, try restarting your App, the "Orkestrator" App and lastly the whole server *in that order*.
Arkitekt's apps are designed to be stateless, so restarting them should not cause any issues. Your data that was stored on the platform should not be lost.
