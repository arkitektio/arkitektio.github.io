# First Steps

This is the first steps tutorial to get you started with your Arkitekt Platform.
It will guide you through the process of connecting your first apps, introduces
basic concepts and shows you how to create your first Arkitekt workflows.

:::info

This tutorial assumes that you have already installed the Arkitekt Platform. If you haven't, please refer to the [Installation](/docs/introduction/installation) section.

:::
