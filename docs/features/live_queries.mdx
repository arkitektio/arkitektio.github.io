# GraphQL

This paragraph shoud explain and give you and idea about why you should use GraphQL.

```jsx live
function Display(props) {
  const { data } = useMikroQuery(
    gql`
      query {
        representations(limit: 3) {
          id
          name
          store
          shape
        }
      }
    `
  );

  return (
    <div className="flex flex-col gap-2">
      {data && data.representations.map((image) => <BioImage image={image} />)}
    </div>
  );
}
```
